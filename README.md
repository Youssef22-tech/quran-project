### Quran Project
A full###stack web application designed to help users organize and track their Quran memorization journey in a structured and interactive way.

### Overview
This project provides a complete system for users to manage daily Quran memorization plans, review progress, and practice revision through an interactive interface. It also includes additional spiritual features such as morning and evening athkar.

### Features

### Authentication System
### User registration and login
### Session###based authentication
### Secure user sessions

### Daily Quran Plans
### Create personalized memorization plans
### Automatic division of the Quran into daily portions
### Track daily progress and completion status

### Memorization & Review
### Display selected verses for memorization
### Hide/reveal text to assist with testing recall
### Resume from last position

### Progress Tracking
### Monitor completion streaks
### Track user progress across days
### Visual indicators for milestones

### Athkar Section
### Morning Athkar
### Evening Athkar
### Organized and easy###to###navigate interface

### Dashboard
### Overview of user progress
### Quick access to active plans
### Summary of achievements

### Tech Stack
### Backend: Node.js, Express.js
### Database: MongoDB with Mongoose
### Frontend: EJS (with ejs###mate layout engine)
### Session Management: express###session
### Flash Messages: connect###flash
### Environment Variables: dotenv

### Getting Started
1### git clone https://github.com/Youssef22###tech/quran###project.git
2### cd quran###project
3### npm install
4### Create a .env file in the root directory:
MONGO_URI=your_mongodb_connection_string
SESSION_SECRET=your_secret_key
PORT=3000
4. npm start
Open http://localhost:3000
